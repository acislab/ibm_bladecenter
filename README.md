# ibm_bladecenter

Scripts for setting up IBM BladeCenters in Python. Mostly for working with
HS22's in H chassis. Beware lots of hard coded stuff.


likely process

1. ./bc_boot_cd.py
2. ./bc_config_blade.py
3. ./bc_boot_diag.py
4. ./bc_boot_net.py
5. (optional) ./bc_enable_quadport.py
6. (or) ./bc_boot_revert.py
